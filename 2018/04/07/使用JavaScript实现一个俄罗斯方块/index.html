иҝҷйҮҢжҳҜжёёжҲҸзҡ„ж ёеҝғйҖ»иҫ‘жүҖеңЁдҪҚзҪ®гҖӮ

еғҸдҝ„зҪ—ж–Ҝж–№еқ—иҝҷз§Қзҡ„зҹ©йҳөзұ»жёёжҲҸпјҢеӯҳеӮЁж•°жҚ®жңҖеҗҲйҖӮзҡ„ж–№жі•е°ұжҳҜдёҖдёӘдәҢз»ҙж•°з»„дәҶгҖӮ

иҖҢдё”иҝҷж ·йҖүжӢ©еңЁдёҖдәӣйҖ»иҫ‘еӨ„зҗҶдёҠд№ҹдјҡжӣҙж–№дҫҝдёҖдәӣпјҡ

  1. дёӢз§»ж“ҚдҪңж—¶пјҢжҲ‘д»¬еҸӘйңҖж”№еҸҳе…ғзҙ зҡ„з¬¬дёҖеұӮдёӢж Ү
  2. еҲӨж–ӯжҳҜеҗҰи§Ұеә•ж—¶пјҢжҲ‘д»¬еҸӘйңҖе°ҶеҪ“еүҚдёӢж Ү + 1 еҲӨж–ӯжҳҜеҗҰжңүе…ғзҙ еҚіеҸҜ

жҲ‘д»¬еҜ№ж•°з»„дёӯзҡ„е…ғзҙ иҝӣиЎҢдәҶе®ҡд№үпјҡ

  • 0: з©әпјҢиЎЁзӨәеҪ“еүҚеқҗж Үдёәз©әзҷҪ
  • 1: ж–°зҡ„ж–№еқ—пјҢиЎЁзӨәеҪ“еүҚжҙ»еҠЁзҡ„ж–№еқ—
  • 2: иҖҒзҡ„ж–№еқ—пјҢе·Із»Ҹи§Ұеә•еӣәе®ҡзҡ„ж–№еқ—

жҺҘдёӢжқҘпјҢжҲ‘д»¬е°ұйҒҮеҲ°дәҶдёҖдёӘй—®йўҳпјҢеҰӮдҪ•еӨ„зҗҶж–№еқ—зҡ„ж”ҫзҪ®гҖӮ
жҲ‘д»¬зҹҘйҒ“пјҢжёёжҲҸдјҡдёҚеҒңзҡ„еҗ‘жЈӢзӣҳдёӯеҠ иҪҪж–°зҡ„ж–№еқ—гҖӮ
еҰӮжһңжҲ‘д»¬жҜҸж¬ЎеӨ„зҗҶдёӢз§»зҡ„ж—¶еҖҷпјҢйғҪе°ҶеҪ“еүҚдәҢз»ҙж•°з»„дёӯеҜ№еә”зҡ„ж–№еқ—е…ғзҙ з§»йҷӨпјҢз„¶еҗҺеңЁеЎһе…ҘеҲ°ж–°зҡ„дҪҚзҪ®пјҢжңӘе…ҚеӨӘиҝҮз№ҒзҗҗдәҶгҖӮ

жүҖд»ҘжҲ‘д»¬еңЁеҲқе§ӢеҢ–ж•°жҚ®ж—¶пјҢеҲқе§ӢеҢ–дёӨдёӘдәҢз»ҙж•°з»„гҖӮ
еҪ“жҲ‘д»¬еҠ иҪҪдёҖдёӘж–°зҡ„ж–№еқ—еҗҺпјҢе°Ҷж–№еқ—еҜ№еә”зҡ„е…ғзҙ еЎһе…Ҙе…¶дёӯзҡ„дёҖдёӘдәҢз»ҙж•°з»„гҖӮ
з„¶еҗҺзӯүеҲ°жҲ‘д»¬жңүиҝӣиЎҢе…¶д»–зҡ„ж“ҚдҪңж—¶пјҢжҜ”еҰӮе·ҰеҸіз§»еҠЁпјҢеҗ‘дёӢд№Ӣзұ»зҡ„гҖӮ
жҲ‘д»¬зӣҙжҺҘдҪҝз”Ёз¬¬дәҢдёӘдәҢз»ҙж•°з»„иҰҶзӣ–еҲ°еҪ“еүҚзҡ„ж•°з»„дёӯеҺ»пјҢз„¶еҗҺеҶҚе°Ҷжӣҙж”№дёӢж ҮеҗҺзҡ„ж–№еқ—еЎһе…Ҙж•°з»„гҖӮ
иҝҷж ·еңЁж•°жҚ®дёҠпјҢжҲ‘д»¬е°ұе®ҢжҲҗдәҶж–№еқ—зҡ„з§»еҠЁгҖӮ

class Game {
  init () {
    // еҲқе§ӢеҢ–дёӨдёӘзҹ©йҳө
    this.matrix = [[], []]
    this.oldMatrix = [[], []]
  }
  move () {
    // йҮҚзҪ®еҪ“еүҚзҹ©йҳөж•°жҚ®
    this.matrix = deepCopy(this.oldMatrix) // и§ЈйҷӨеј•з”Ё
    // еҠ иҪҪж–№еқ—ж•°жҚ®
    this.matrix[y][x1] = 1
    this.matrix[y][x2] = 1
  }
}

е·ҰеҸіз§»еҠЁзҡ„еӨ„зҗҶ

е·ҰеҸізҡ„з§»еҠЁдёҚиғҪеғҸеҗ‘дёӢз§»еҠЁдёҖж ·пјҢеҚ•зәҜзҡ„дёӢж Ү+1гҖӮ
жҲ‘д»¬йңҖиҰҒеҲӨж–ӯеҪ“еүҚзҡ„ж“ҚдҪңжҳҜеҗҰжңүж•ҲгҖӮ
жҜ”еҰӮеҸідҫ§еҰӮжһңйҒҮеҲ°дәҶйҡңзўҚзү©жҲ–иҖ…еҲ°иҫҫиҫ№зјҳпјҢжҲ‘д»¬иӮҜе®ҡжҳҜдёҚиғҪеӨҹеҶҚиҝӣиЎҢз§»еҠЁзҡ„гҖӮ

// blend дёәжҙ»еҠЁз –еқ—зҡ„еҪўзҠ¶жҸҸиҝ° [[1, 1, 1], [0, 1, 0]] зұ»дјјиҝҷж ·зҡ„з»“жһ„
if (
  x >= width - brickWidth ||
  blend.some((row, rowIndex) => {
    let _pos = oldMatrix[y + rowIndex]
    return row && row[brickWidth - 1] && _pos && _pos[x + brickWidth]
  })
)
  return // еҸідҫ§жңүйҡңзўҚзү©пјҢж— жі•з§»еҠЁ

дҪҝз”Ёзұ»дјјиҝҷж ·зҡ„йҖ»иҫ‘иҝӣиЎҢеҲӨж–ӯпјҢдҝқиҜҒеҪ“еүҚж–№еқ—еҗ‘еҸіз§»еҠЁеҗҺдёҚдјҡиҰҶзӣ–д№ӢеүҚзҡ„ж–№еқ—гҖӮ

еҝ«йҖҹеҗ‘дёӢзҡ„еӨ„зҗҶ

жҲ‘зңӢжңүдәӣжёёжҲҸе®һзҺ°зҡ„пјҢиІҢдјјдёӢйҷҚи§ҰеҸ‘еҸӘжҳҜеҠ йҖҹдёӢйҷҚиҖҢе·ІпјҲиҝҷз§Қжғ…еҶөеҸӘйңҖиҰҒж”№еҸҳе®ҡж—¶дёӢйҷҚзҡ„йҖҹеәҰеҚіеҸҜпјү-.-иҝҷйҮҢзҡ„е®һзҺ°жҳҜпјҢзӣҙжҺҘи§Ұеә•

жүҖд»Ҙе°ұдјҡйҒҮеҲ°дёҖдёӘй—®йўҳпјҢеҪ“еүҚз –еқ—жңҖеӨҡеҸҜд»ҘдёӢйҷҚеҲ°д»Җд№ҲдҪҚзҪ®пјҹ

е°ұеғҸиҝҷж ·зҡ„дёҖдёӘж•°жҚ®пјҢ0|2иҝҷдёӨеҲ—йғҪеҸҜд»Ҙеҗ‘дёӢз§»еҠЁдёӨеҲ—пјҢдҪҶжҳҜиҝҷж ·е°ұдјҡеҜјиҮҙдёӯй—ҙдёҖеҲ—зҡ„йҮҚеҸ гҖӮ
жҲ‘д»¬дёҖе®ҡиҰҒеҸ–еҮәдёӢйҷҚе№…еәҰжңҖе°Ҹзҡ„йӮЈдёӘеҖјгҖӮ
жүҖд»ҘжҲ‘д»¬е°ұиҰҒз®—еҮәжңҖеҗҺдёҖиЎҢ1зҡ„дёӢж Үд»ҘеҸҠз¬¬дёҖиЎҢ2зҡ„дёӢж ҮпјҢе°ҶиҝҷдёӨдёӘдёӢж ҮиҝӣиЎҢзӣёеҮҸпјҢжңҖе°ҸеҖјеҚідёәжҲ‘д»¬еҪ“еүҚж–№еқ—еҸҜдёӢйҷҚзҡ„и·қзҰ»гҖӮ

ж—ӢиҪ¬ж–№еқ—зҡ„еӨ„зҗҶ

ж—ӢиҪ¬ж–№еқ—еә”иҜҘжҳҜжёёжҲҸдёӯжҜ”иҫғеӨҚжқӮзҡ„дёҖеқ—йҖ»иҫ‘дәҶгҖӮ
з»қдёҚжҳҜд»…д»…з®ҖеҚ•зҡ„е°Ҷж–№еқ—зҡ„дәҢз»ҙж•°з»„з”ұиЎҢж”№дёәеҲ—пјҢеңЁжңүдәӣж—¶еҖҷпјҢжҲ‘д»¬иҝҳйңҖиҰҒеҲӨж–ӯж–№еқ—жҳҜеҗҰеҸҜд»ҘиҝӣиЎҢж—ӢиҪ¬гҖӮ

е°ұеғҸиҝҷж ·зҡ„пјҢдёӯй—ҙзҡ„й•ҝжқЎжҳҜдёҚиғҪеӨҹиҝӣиЎҢж—ӢиҪ¬зҡ„гҖӮ
жүҖд»ҘжҲ‘д»¬иҰҒе…ҲжӢҝеҲ°ж—ӢиҪ¬еҗҺзҡ„ж•°жҚ®пјҢжқҘдёҺеҪ“еүҚжёёжҲҸдёӯзҡ„ж•°жҚ®иҝӣиЎҢжҜ”иҫғпјҢжЈҖйӘҢжҳҜеҗҰдјҡеҮәзҺ°йҮҚеҸ зҡ„жғ…еҶөпјҢеҰӮжһңеҮәзҺ°дәҶпјҢеҲҷиЎЁзӨәдёҚиғҪеӨҹиҝӣиЎҢж—ӢиҪ¬гҖӮ

и§Ұеә•жЈҖжөӢ

жҜҸе®ҢжҲҗдёҖдёӘз§»еҠЁзҡ„еҠЁдҪңеҗҺпјҢжҲ‘д»¬йғҪйңҖиҰҒиҝӣиЎҢж–№еқ—зҡ„и§Ұеә•жЈҖжөӢгҖӮ
д№ҹе°ұжҳҜеҲӨж–ӯеҪ“еүҚж–№еқ—дёӢпјҢжҳҜеҗҰе·Із»Ҹжңүе…ғзҙ еҚ дҪҚпјҢеҰӮжһңжңүзҡ„иҜқпјҢеҲҷиЎЁзӨәе·Із»Ҹи§Ұеә•дәҶпјҢеҪ“еүҚе…ғзҙ е°ұдјҡиў«еӣәе®ҡиҝӣзҹ©йҳөж•°з»„дёӯгҖӮ
еҗҢж ·зҡ„пјҢжҲ‘д»¬еңЁеҲӨж–ӯж—¶пјҢдёҚйңҖиҰҒе°Ҷж–№еқ—жүҖжңүзҡ„дёӢж ҮйғҪжЈҖжҹҘдёҖйҒҚпјҢеҸӘйңҖиҰҒжЈҖжҹҘжңҖеә•йғЁдёҖеұӮзҡ„жңүж•Ҳе…ғзҙ еҚіеҸҜгҖӮ

еғҸиҝҷж ·зҡ„дёҖдёӘж–№еқ—пјҢжҲ‘д»¬д»…йңҖиҰҒеҲӨж–ӯз¬¬дёҖеҲ—зҡ„з¬¬дәҢиЎҢ&з¬¬дәҢеҲ—зҡ„з¬¬еӣӣиЎҢжҳҜеҗҰжңүе…ғзҙ еҚіеҸҜе®ҢжҲҗжЈҖжҹҘгҖӮ

з§»йҷӨиЎҢ

еңЁи§Ұеә•жЈҖжөӢи§ҰеҸ‘еҗҺпјҢеҰӮжһңжңүж–№еқ—иў«еӣәе®ҡиҝӣж•°з»„пјҢжӯӨж—¶жҲ‘д»¬еҶҚиҝӣиЎҢз§»йҷӨиЎҢзҡ„ж“ҚдҪңгҖӮ
